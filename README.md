## 🚀 Getting Started

### Prerequisites
- Python 3.10 or 3.11
- Poetry (for dependency management)

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/ML_Projects.git
cd ML_Projects

# Install dependencies using Poetry
poetry install

# Activate the virtual environment
poetry shell

# Start Jupyter Lab
jupyter lab
```
